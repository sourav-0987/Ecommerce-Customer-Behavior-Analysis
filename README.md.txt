# E-Commerce Customer Behavior Analysis

## Overview
This project analyzes customer purchasing behavior, revenue trends, payment preferences, and customer retention using SQL.

## Tools Used
- MySQL
- SQL
- MySQL Workbench
- VS Code

## Database Schema
- customers
- products
- orders
- order_items
- payments

## SQL Concepts Used
- Joins
- Aggregate Functions
- GROUP BY
- HAVING
- Common Table Expressions (CTEs)
- Foreign Keys

## Business Questions Solved
1. Top Customers by Spending
2. Monthly Revenue Analysis
3. Product Category Analysis
4. Payment Method Analysis
5. Repeat Customers
6. Customer Retention Rate

## Key Insights
- Repeat customers contribute significantly to revenue.
- Electronics products drive most sales.
- UPI is the most frequently used payment method.
- Monthly revenue trends help identify sales patterns.
